3.1.2 The gnun-add-fuzzy-diff Script

This script adds comments with differences of current msgids against previous ones to fuzzy translations in a PO file. To produce the differences wdiff is used. This may be useful to figure out what has changed. In fact, it wraps around a sed script used in GNUN internally. 

     gnun-add-fuzzy-diff [OPTION...] [FILE]
-i
--in-place
Edit the file in place.
--version
Display copyright and version information and exit.
--help
Display usage information and exit
