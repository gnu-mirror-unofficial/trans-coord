2.1.2.2 The update-localized-URLs Target

The update-localized-URLs target invokes a script that generates the localized-urls.mk file. This file includes the list of URLs that should be localized, it is extracted from special comments in the HTML source (see Localized URLs).

This target is meant to be rebuilt nightly.
