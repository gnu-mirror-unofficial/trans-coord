6.2 GNUN Scripts

For the time being there are several helper scripts. Some of them are used internally as commands with certain arguments in the makefile rules; other scripts were written specifically to facilitate some mechanical actions, like initially filling the headers in the PO files. They all can be invoked separately, as stand-alone programs.

• gnun-add-fuzzy-diff:  Add differences to “previous” msgids.
• gnun-diff-po:  Compare two revisions of a PO file.
• gnun-init-po:  Initialize a new translation.
• gnun-preconvert:  Invoke the first step in converting HTML translation to PO format.
• gnun-merge-preconverted:  Run the second step of the conversion.
• gnun-report:  Generate report in HTML format.
• gnun-validate-html:  Validate HTML file.
