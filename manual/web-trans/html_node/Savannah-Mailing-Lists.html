3.2.3.8 Managing Mailing Lists

You can create new mailing lists via the Savannah interface. However, this should be done after some thought. If the project membership is low (<= 10 members), there is no need to create more than one mailing list. If there is a general translation list common for all translators of the language (i.e. GNOME, GNUstep, KDE, and free software in general), there is no reason at all to create a list at Savannah.

You can redirect all messages generated by the trackers to any list (including those that are not on GNU servers).
