3.5.5 Bugs Tracker

The ‘Bugs’ tracker is designed for tracking bugs. You can use for several purposes:

  • Suggest readers to report bugs there.
  • Use it for all kinds of internal team tasks.
  • Forward bugs reported against lang translations in the ‘trans-coord’ project and assign them to the specific maintainer (who is supposed to be a ‘www-lang’ project member), if you have such policy.
